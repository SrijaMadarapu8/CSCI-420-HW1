To run the code :
for one image : .\hw1.exe imagefilename.jpg
for color values taken from another image of equal size : .\hw1.exe imagefilename.jpg imagefile1.jpg

Cases:

1 : To display points
! : To display points using colors from jetmapcolor function
2 : To display lines
@ : To display lines using colors from jetmapcolor function
3 : To display triangles
# : To display triangles using colors from jetmapcolor function
4 : To display Smooth triangles
$ : To display Smooth triangles using colors from jetmapcolor function
5 : colored wireframe on top of solid triangles
+/- : To scale the smooth triangle
0/9 : To exponent the smooth triangle
a : for animation and taking screenshots
s : for animation and taking screenshots for color values taken from another image mode

Modes in shader:

1 : for smooth triangles
 grayscale=1 : gray values is rendered
 grayscale=0 : color values is rendered
2 : for color values in other cases like points lines triangles
